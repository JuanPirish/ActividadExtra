# ActividadExtra
Carpetas por unidades y resumen

La licencia MIT en repositorios de GitHub es ideal para fomentar el uso generalizado y
la colaboración, ya que es permisiva, simple y breve. Permite a otros desarrolladores
usar, modificar, distribuir y vender el código sin restricciones, incluso en proyectos
propietarios, manteniendo al autor original sin responsabilidad y sin ofrecer
garantías.
